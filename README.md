# Minimal Monochrome Conky Widget

A minimal, compact, and semi-transparent Conky widget for Linux desktop.
Designed for clean system monitoring without visual clutter.

Tested on:
- Linux Mint
- Ubuntu
- Cinnamon, XFCE, Mate

---

## Features

- CPU usage with bar and graph
- CPU temperature
- Real RAM usage (used / total)
- Disk usage
- Network upload & download speed
- Monochrome gray color scheme
- Compact and clean layout
- Stable on Cinnamon (no X11 crash)

---

## Requirements

Install required packages:

```bash
sudo apt update
sudo apt install conky-all lm-sensors -y
